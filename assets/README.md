# Assets do VendaNinja

Este diretório contém os arquivos de mídia necessários para o VendaNinja.

## Arquivos Necessários

### shuriken.mp3
- **Descrição**: Som de shuriken sendo lançado, reproduzido quando um item é adicionado ao carrinho
- **Formato**: MP3
- **Duração**: ~0.5 segundos
- **Como obter**: 
  - Use um editor de áudio para criar um som curto de "whoosh" ou "swish"
  - Ou baixe um efeito sonoro gratuito de sites como freesound.org
  - Ou remova a funcionalidade de som editando script.js

### logo-ninja.png
- **Descrição**: Logo do VendaNinja com tema de ninja
- **Formato**: PNG
- **Tamanho recomendado**: 200x200px ou maior (retangular também funciona)
- **Como obter**:
  - Crie um logo simples com um ícone de ninja usando ferramentas como Canva, Figma, ou GIMP
  - Ou use um ícone de ninja de sites como flaticon.com ou iconfinder.com
  - O sistema funcionará mesmo sem este arquivo (ele será ocultado se não existir)

### icon-192.png
- **Descrição**: Ícone do PWA para instalação no dispositivo
- **Formato**: PNG
- **Tamanho**: 192x192px (e 512x512px para melhor qualidade)
- **Como obter**:
  - Use a mesma imagem do logo-ninja.png redimensionada para 192x192px
  - Ou crie um ícone específico usando ferramentas online de geração de ícones PWA
  - O ícone deve ser quadrado e ter boa visibilidade em tamanhos pequenos

## Nota

Se você não tiver estes arquivos, o sistema ainda funcionará normalmente. O som simplesmente não será reproduzido e o logo será ocultado automaticamente.
